Hacking the Matrix

Nero is trying to hack the Matrix. He has a list of codes, but to solve this challenge, he must rewrite the map with the length of the code as a key and the code as value. Please help Nero by printing the map to hack the Matrix.


Sample Input:
1234mom daddy4456 god hack me

Sample Output:
{7=1234mom, 9=daddy4456, 3=god, 4=hack, 2=me}



Memory limit: 256 MB
Time limit: 8 seconds
