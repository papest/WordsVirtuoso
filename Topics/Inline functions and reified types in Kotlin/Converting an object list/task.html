Converting an object list

Converting a list of objects of one type to a list of objects of another type is quite common. Write a universal convertList function that can convert a list of objects from one type to another using a function passed as a parameter.

In your function, use the two reified type parameters – T and R. T represents the type of elements in the input list, and R represents the type of elements in the output list. The transform function takes an element of type T and converts it to an element of type R.



Memory limit: 256 MB
Time limit: 8 seconds

Caution

You may see errors in your code or execution results due to missing context. Don’t worry about it, just write the solution and press Check.
